# HUNT

### Description:
2D side scroller action-platformer game. The objective is to go through levels and kill enemy aliens.

### Platform:
Java.

### Tools used:
* Programmed in JAVA.
* Slick2D used as graphics api.

### Authors:
1. Md. Fahim Mohiuddin. (program) 
2. Fayza Amreen. (asset design & level design)
